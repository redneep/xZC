Настройка
xZC

Подробное руководство по настройке скрипта с нуля. Выберите платформу авторизации и следуйте шагам.

🎉 Поздравляем с получением xZC! Для запуска скрипта выполните несколько простых шагов ниже.
шаг 01

Найдите BotFather в Telegram

01

Найдите BotFather

Откройте Telegram и в поиске найдите бота @BotFather

шаг 02

Начните диалог с ботом

02

Нажмите Start

Нажмите кнопку «Start» или «Open» для начала работы с BotFather

шаг 03

Создайте нового бота

03

Create a New Bot

В меню бота выберите опцию «Create a New Bot»

шаг 04

Настройте имя бота и получите токен

04

Придумайте имя и юзернейм

Придумайте название бота (Bot Name) и юзернейм (username). Юзернейм должен оканчиваться на bot или _bot.

💡
Важно: Следуйте инструкциям на видео внимательно!
⚠️
Если токен не скопировался автоматически: Следуйте инструкции ниже. Если токен скопировался успешно — переходите к шагу 5.
шаг 05

Вставьте токен в скрипт

05

Вставьте токен

Вставьте полученный токен в указанное место в скрипте, используя Ctrl+V

шаг 06

Найдите свой Telegram ID

06

Настройки профиля

Зайдите в настройки профиля Telegram как показано на изображении

шаг 07

Скопируйте цифровой ID

07

Нажмите на ID

Найдите строку с цифровым идентификатором (синие цифры) и нажмите на неё для копирования

шаг 08

Вставьте TG ID и сохраните

08

Вставить и сохранить

Вставьте скопированный ID в указанное место и нажмите кнопку «Сохранить»

шаг 01

Создайте сообщество ВКонтакте

01

Раздел «Сообщества»

Зайдите в раздел «Сообщества» и создайте новое сообщество

шаг 02

Настройки сообщества

02

Настройки

Как вы его настроете — совершенно не важно, это никак не влияет

шаг 03

Управление

03

Нажмите «Управление»

Настроим сообщество полностью — нажмите «Управление»

шаг 04

Включите бота

04

Возможности ботов

Перейдите: Сообщения → Настройки для бота → Возможности ботов

По желанию можно включить «Добавить кнопку Начать»

шаг 05

Создайте API-ключ и настройте Long Poll

05a

Включаем API

Перейдите: Дополнительно → Работа с API → Создать ключ

Чтобы не разбираться — ставьте все галочки. После создания скопируйте ключ и вставьте в скрипт в строку Токен бота (TG) / VK Token.

🔒
Никогда не давайте API-ключ посторонним, особенно если сообщество не анонимное!
05b

Long Poll API

Перейдите в Long Poll API (находится рядом с «Создать ключ») → включите Long Poll API → «Тип событий»

В разделе Сообщения поставьте все галочки. Остальные разделы можно не отмечать.

шаг 06

Узнайте свой VK ID

06

Скопируйте URL профиля

Зайдите в свой профиль, скопируйте URL из адресной строки

шаг 07

Получите числовой ID

07

regvk.com/id

Перейдите на regvk.com/id и вставьте скопированный URL. Скопируйте «ID пользователя».

шаг 08

Вставьте ID и сохраните

08

Сохранить

Вставьте скопированный VK ID в указанное поле и нажмите «Сохранить»

шаг 09

ID сообщества VK

09

Скопируйте ID сообщества

Вернитесь в созданное сообщество. В адресной строке будет что-то вроде https://vk.ru/club000000001 — вам нужны только цифры: 000000001. Это и есть ID сообщества. Скопируйте и вставьте в нужное поле.
